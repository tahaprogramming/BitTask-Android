# BitTask

A professional, offline-first To-Do application built with Kotlin, Jetpack Compose, and Material Design 3. 

## Features

- **Offline-First:** Designed to work entirely without an internet connection.
- **Material Design 3:** Modern, accessible, and adaptive UI.
- **Jalali/Gregorian Calendar:** Supports multiple calendar systems for your planning needs.
- **Local Persistence:** Data is stored locally using Room, ensuring privacy and speed.
- **Smart Notifications:** Get reminded of your upcoming tasks with ease.
- **Sub-task Management:** Organize your tasks with checklists.

## Technical Architecture

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Persistence:** Room Database
- **Asynchronous Operations:** Coroutines & Flow

## Getting Started

1. Clone the repository.
2. Open in Android Studio.
3. Build and run.
